# IDon'tTrustMyMembers

Cross-server moderation intelligence system for Discord communities.

---

## Overview

IDon'tTrustMyMembers is a distributed moderation intelligence network for Discord servers.

It enables shared security data across multiple communities.

---

## Features

→ Cross-server ban intelligence  
→ User reputation scoring  
→ Global lookup system  
→ Real-time synchronization  
→ Automatic join analysis  
→ SQLite-based storage  

---

## How it works

1. A server adds the bot  
2. Existing bans are indexed  
3. New moderation actions are synced  
4. Users can be looked up globally  
5. Risk score is calculated per user
